To run MLEM2 program first type make. Then type ./MLEM2Prog and then you will be prompted for file names.
With certain execution commands the output file name will be appended with .out.txt to facilitate cleanup 
with makefile.

Main option for running MLEM2Prog:
./MLEM2Prog                    : this will prompt you for filenames of your selection
Other options for running MLEM2Prog:

./MLEM2Prog <infile>           : this will generate an outfile of <infile> + ".out.txt"
./MLEM2Prog <infile> <outfile> : this will generate an outfile of <outfile> + ".out.txt"
